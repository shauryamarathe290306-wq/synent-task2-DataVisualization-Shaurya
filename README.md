# 📖 Project Overview

This project is part of the **Synent Technologies Data Science Internship**. The objective is to explore the Iris dataset through meaningful visualizations that help identify patterns, relationships, and distributions among different Iris flower species.

Data visualization is an important step in the data analysis process because it transforms raw data into graphical representations that are easy to understand and interpret.

---

# 🎯 Problem Statement

The Iris dataset contains measurements of three species of Iris flowers: **Setosa**, **Versicolor**, and **Virginica**. The challenge is to visualize the dataset using different chart types to understand feature distributions and relationships between variables.

The project aims to create informative visualizations that support data exploration and provide meaningful insights.

---

# ✅ Task Explanation

The primary objective of this task is to perform data visualization on the Iris dataset.

The following visualizations were created:

### 📊 Bar Chart
- Displays the number of samples for each Iris species.
- Helps compare the distribution of flower species.

### 📈 Histogram
- Shows the distribution of Sepal Length.
- Helps understand the spread and frequency of values.

### 🔵 Scatter Plot
- Compares Sepal Length and Petal Length.
- Uses different colors to represent each Iris species.
- Helps identify relationships and clustering among the species.

Each visualization includes appropriate titles, axis labels, legends, and formatting to improve readability and interpretation.

# 📈 Results

The visualizations reveal several important observations:

- The dataset is balanced, containing an equal number of samples for each Iris species.
- Sepal Length follows a moderately distributed pattern with no major outliers.
- The scatter plot clearly separates the Setosa species from the other two species.
- Versicolor and Virginica exhibit some overlap but remain distinguishable.
- Petal Length is an effective feature for differentiating Iris flower species.
